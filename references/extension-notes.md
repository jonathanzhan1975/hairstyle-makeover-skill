# Extension Notes

This skill is intentionally hairstyle-first, but the structure is suitable for broader image makeover work.

## Future expansion areas

- glasses recommendation
- beard shaping
- earrings or accessories
- clothing coordination
- portrait presentation boards

## Keep stable if extended

Even when expanding beyond hair, preserve the same logic:
- analyze first
- define edit boundaries
- rank options
- generate structured prompts
- prioritize same-person consistency

## Do not expand too early

For this first version, hairstyle remains the primary edited attribute.
If the user asks for broader makeover advice, keep the hairstyle recommendation as the anchor and add only one adjacent dimension at a time.
